### hoisting은 무엇인가 
- 선언되지않은 값을 끌어올리는 것

### console log 어떤순서로 출력 되는가
```node
console.log("Start");

setTimeout(() => {
  console.log("Timeout");
}, 1000);

Promise.resolve().then(() => {
  console.log("Promise");
});

console.log("End");

// "Start"
// "End"
// "Promise"
// "Timeout"
```
### ==과 ===의 차이
- Null == undefined 그리고 null === undefined 결과
```node
console.log(null == undefined)
//true
console.log(null === undefined)
//false
```

### Null == false
```node
console.log(null==false)
//false
```
